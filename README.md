| `formacao.html` | Formação — linha do tempo, habilidades e idiomas |

| `contato.html` | Contato — formulário com validação JavaScript |

---

## ⚙️ Tecnologias Utilizadas

- **HTML5** — Estrutura semântica com doctype HTML5
- **CSS3 Puro** — Sem frameworks (sem Bootstrap, sem Tailwind)
- **JavaScript Puro** — Sem jQuery, React ou qualquer biblioteca

> ⚠️ Conforme exigido pela atividade, **nenhum framework** foi utilizado.

---

## ✅ Funcionalidades JavaScript

- 🌙 **Alternância de tema dark/light** com persistência via `localStorage`
- 📱 **Menu hamburguer responsivo** para dispositivos móveis
- 🔗 **Marcação de link ativo** no menu de navegação
- 📊 **Animação das barras de habilidade** via `IntersectionObserver`
- 🔍 **Filtro de projetos** por categoria no portfólio
- ✅ **Validação completa do formulário** (nome, e-mail, assunto, mensagem)
- 🎉 **Modal de confirmação** após envio bem-sucedido
- 💬 **Contador de caracteres** em tempo real na textarea
